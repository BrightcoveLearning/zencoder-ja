意味

入力ファイルに対して計算された MD5 チェックサムが API に提供されたチェックサムと一致しません。

この問題には、さまざまな原因が考えられます。最も可能性の高いものは次のとおりです。

  • ファイルは転送中に破損しています。
  • API に提供されたチェックサムが正しくありません。

回避策

API に提供したチェックサムが、ディスク上のファイルのチェックサムと一致することを確認します。

もし彼らが しないでください 一致する場合は、適切なチェックサムを使用して新しいAPI呼び出しを行ってください。

チェックサムの場合 行う 一致する場合、ファイルは転送中に破損している可能性があります。ジョブを再送信してください。

いずれの場合も、再試行に失敗した場合は、お問い合わせください。

このジョブを再試行しますか？

このエラーのあるジョブは再試行できます。ジョブを再試行する方法の詳細をご覧ください。
