意味

このエラーは、同じストリーム ID を使用するストリームが多すぎるため、CDN が RTMP ストリームを拒否したことを示します。したがって、IDあたりのストリームの最大数を超えています。

詳細情報

これは、次のことを意味する可能性があります。

  • CDN で設定された ID あたりの最大ストリーム数が小さすぎて、要求されたストリームをすべて受け入れることができません。
  • 他の Zencoder ジョブはすでにこの CDN とストリーム ID にストリーミングしています
  • CDN ストリーム ID は Zencoder 以外の別のシステムで使用されています
  • CDN には「' ゾンビストリーム」があり、以前のストリームが適切に切断されておらず、この ID の最大ストリーム制限にカウントされます。

この問題を回避するには、このストリーム ID を使用してストリームを持つ他のジョブを停止するか、別のストリーム ID がある場合は別のストリーム ID を使用します。詳細については、Job ID でサポートに連絡してください。当社のエンジニアリングチームが調査します。

このジョブを再試行しますか？

このエラーのあるジョブは再試行しないでください。ジョブを再試行する方法の詳細をご覧ください。
