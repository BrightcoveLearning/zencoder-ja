意味

指定されたアドレスでファイルをダウンロードしようとしましたが、許可エラーが発生しました。これが S3 URL の場合は、おそらくこのバケットまたはファイルの Zencoder への読み取りアクセスを許可する必要があります。これを行う最も簡単な方法は、Amazon の AWS コンソールのように S3 クライアントを使用することです。

技術的な話では、これは、サーバーから403アクセスが拒否された応答を得ていることを意味します。

回避策

指定された URL のファイルに対するパーミッションが Zencoder アクセスを許可していることを確認してください。ファイルが S3 上にある場合は、Zencoder への読み取りアクセスを許可する必要があります。
