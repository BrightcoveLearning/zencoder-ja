{{ page.title }}

{{ page.description }}

意味

指定されたアドレスでサーバーに接続しようとしましたが、サーバーが応答しませんでした。これは、サーバーがファイアウォールの内側にある場合、アクセスが特定の IP アドレスに制限されている場合、またはサーバーがオフラインである場合に発生します。

回避策

サーバーがオンラインであり、外部要求に対してパブリックにアクセス可能であることを確認してください。サーバーへのアクセスが特定の IP に制限されている場合は、Zencoder が使用する IP 範囲について詳しく知ることができます。

このジョブを再試行しますか？

このエラーのあるジョブは再試行できます。ジョブを再試行する方法の詳細をご覧ください。
