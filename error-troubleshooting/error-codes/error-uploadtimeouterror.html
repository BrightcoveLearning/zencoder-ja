    UploadTimeouTerror

    このトピックでは、Zencoder API エラーの詳細について説明しますUploadTimeoutError。

    意味

    指定されたアドレスにファイルをアップロードしようとしましたが、そのアドレスのサーバーは応答に時間がかかりすぎます。この問題は、一時的な接続の問題ではないことを確認するために、ファイルを 3 回ダウンロードしようとします。

    このジョブを再試行しますか？

    このエラーのあるジョブは再試行しないでください。ジョブの再試行の詳細については、こちらをご覧ください。
