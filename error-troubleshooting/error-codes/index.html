エラーコード

このトピックでは、Zencoder API リクエストの HTTP ステータスコードの概要と、Zencoder API によって返されるエラーコードのインデックスについて説明します。エラーコードをクリックすると、詳細が表示されます。

HTTP ステータスコード

次の表に、Zencoder API リクエストの HTTP レスポンスコードを示します。エラーの場合、考えられる原因は可能な限り示されています。

HTTP レスポンスコード
ステータスコード 考えられる説明
200 OK 新しいアセットを作成しないアクションに対する標準の成功レスポンス
201 CREATED 新しいアセットを作成するリクエストに対する標準対応
204 NO CONTENT 応答を返さなかったリクエストの標準応答
400 BAD REQUEST リクエスト本文に無効な JSON
401 UNAUTHORIZED API キーがないか無効です
402 PAYMENT REQUIRED API リクエストが、課金が設定されていないアカウントで Zencoder 統合モード https://app.zencoder.com/docs/api/accounts/integration を無効にしようとしたときに返されます。www.zencoder.comにアクセスして支払いプランを選択してください
403 FORBIDDEN リクエストのレート制限を超過した、または何度も再試行されたリソースに対する再試行/再送信 API リクエストへの応答として
404 NOT FOUND リクエストURLが正しくないか、無効/古いジョブ ID
406 NOT ACCEPTIBLE API を使用して AWS 以外の認証情報を作成しようとした場合に返されます。
409 CONFLICT 一般的に、古すぎるジョブを再試行/再提出しようとするときのみ
422 UNPROCESSIBLE ENTITY API リクエストを検証できない場合に返されます
500 INTERNAL SERVER ERROR Zencoder の一時的な問題、またはリクエストに何か問題がある
503 SERVICE UNAVAILABLE Zencoder が一時的に使用できない (おそらく定期メンテナンス中)
504 GATEWAY TIMEOUT Zencoder サービスがダウンしている場合に返されます。
