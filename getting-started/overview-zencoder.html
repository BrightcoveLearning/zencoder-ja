概要:Zencoder

Zencoderは、オンデマンドビデオをトランスコードするための強力なプラットフォームです。

はじめに

Zencoder は、ビデオおよびオーディオファイルの高速かつ信頼性の高いトランスコーディングサービスを提供しています。ウェブやさまざまなデバイス用の出力を生成でき、CMSやその他のアプリケーションと簡単に統合できます。

UI

Zencoder は主に API ですが、以下を含む UI があります。

  • アカウント設定
  • 最近のジョブの結果を示すダッシュボード
  • コードを記述せずに API を使い始めることができる API リクエストビルダー

API

API を使用して実行できるいくつかの関数がありますが、その中で最も重要なのは、ビデオオンデマンドのエンコーディングジョブを送信および管理することです。

API エンドポイントの完全なガイドについては、 API リファレンスを参照してください。

Zencoder の力は、必要な種類の出力を正確に作成できる柔軟性にあります。理解するために多くのエンコーディング設定があるので、これはいくつかの複雑さの源でもあります。これらを調べるのに適した場所は、エンコーディング設定の概要です。ここでは、サンプルコードを含む完全な説明を参照するためのリンクがあります。
