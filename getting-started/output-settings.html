出力設定

このトピックでは、Zencoderジョブのデフォルトの出力と、独自の出力を定義するための基本について説明します。

出力を定義する

デフォルトの出力は、次の設定の単一のビデオファイルです。

H.264 ビデオ、AAC オーディオ、MP4 ファイル形式
ビデオ: 中品質
オーディオ：ステレオ、中品質（約112kbps）、44100 Hz

出力場所を指定しない場合、ファイルは Zencoder によって一時的にホストされます。24 時間が経過すると、ファイルは利用できなくなります。

もちろん、出力設定を制御することもできます。これを行うには、次のように API リクエストでオプションの配列を渡します。

{
  "api_key": "93h630j1dsyshjef620qlkavnmzui3",
  "input": "s3://bucket-name/file-name.avi",
  "outputs": [
    {
      "url": "s3://output-bucket/output-file-name.mp4",
      "width": "1280",
      "height": "720"
    }
  ]
}

この例では、元の入力ビデオから 1 つの出力ファイルを作成し、S3 バケットにアップロードします。追加のエンコーディングジョブを作成しなくても、オリジナルから複数の出力ファイルを作成できます。
