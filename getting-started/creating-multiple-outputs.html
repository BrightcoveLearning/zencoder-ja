複数の出力の作成

このトピックでは、エンコードジョブの複数の出力を作成する方法について説明します。

複数の出力を作成する

次のように outputs 配列に複数のハッシュを渡すことによって、複数の出力を指定できます。

{
  "input": "https://user:pass@bucket-name/file-name.avi",
  "outputs": [
    {
      "label": "web",
      "url": "s3://output-bucket/web.mp4",
      "size": "512x384"
    },
    {
      "label": "iphone",
      "url": "s3://output-bucket/iphone.m4v",
      "size": "480x320"
    }
  ]
}
