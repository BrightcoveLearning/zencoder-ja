概要

AS-11 出力を作成するには、format_presetパラメーターを「as11_hd」に設定します。現在、オーディオは2チャンネルPCMオーディオに制限されており、最大サンプルレートは48KHzです。AVC-INTRA 出力の詳細については、こちらのドキュメントを参照してください。

  • フォーマットプリセット :ae avCi 100 MXF 出力の場合、これをas11_hd に設定します。
  • audio_sample_rate :デフォルトは設定されていません。48000に限る。

設定例

AS-11 リクエストの例

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "format_preset": "as11_hd",
        "audio_sample_rate": 48000
    }
}
