AVC-イントラの概要

AVC-intraは、AVC-Intra50 と AVC-Intra100 の2つのプリセットを定義しています。AVC-イントラ 50 は 50 メガビット/秒のビットレートでエンコードされますが、AVC-イントラ 100 は 100 メガビット/秒でエンコードされます。video_codec_presetパラメータを使用して avCi 出力を定義します。AS-11 出力の作成については、こちらのドキュメントを参照してください。

  • video_codec_preset :出力の AVC-Intra プリセットを設定します。値:avci_50、avci_100。

設定例

AVC-イントラ 50 リクエストの例

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "video_codec_preset": "avci_50"
    }
}

AVC-イントラ 100 リクエストの例

{
    "input": "s3://zencodertesting/test.mov",
    "output": {
        "video_codec_preset": "avci_100"
    }
}
