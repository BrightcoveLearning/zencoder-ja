start_clip

start_clip：String

APIバージョン：V1、V2

親:出力

有効な値：タイムコード ( HH: MM: SS.S )、または秒数を表す正の数値

互換性のあるジョブタイプ： VOD

例:

  • 00:00:10 .0
  • 01:31:08 .3
  • 60
  • 12.5

説明:

特定の時間から開始して、元のビデオのサブクリップを作成します。これは、 HH: MM: SS.S の形式のタイムコード、または開始する秒数を表す小数点のいずれかになります。ビデオ内のキーフレームの配置によっては、実際の開始時間が完全に正確でない場合があることに注意してください。

{ "input": "s3://zencodertesting/test.mov", "outputs": [ { "start_clip": "00:00:10.0" } ]}

参照:clip_length

clip_length

clip_length：String

APIバージョン：V1、V2

親:出力

有効な値：HH: MM: SS.S の形式のデュレーション、または秒数を表す正の数を指定します。

互換性のあるジョブタイプ：すべて

例:

  • 00:00:10 .0
  • 01:31:08 .3
  • 60
  • 12.5

説明:

特定の長さのサブクリップを作成します。これは、 HH: MM: SS.S の形式のデュレーション、またはクリップの長さを秒単位で表す小数点のいずれかになります。start_clipと組み合わせて、ビデオの途中でサブクリップを作成できます。start_clipが指定されていない場合、クリップはビデオの先頭から開始されます。ビデオ内のキーフレームの配置によっては、実際のクリップの長さは完全には正確ではない場合があります。また、clip_lengthは期間であり、サブクリップが終了するメディアファイルのタイムコードではないことに注意してください。

{ "input": "s3://zencodertesting/test.mov", "outputs": [ { "clip_length": "00:00:10.0" } ]}

参照:start_clip
