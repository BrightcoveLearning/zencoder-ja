回転する

rotate:Integer

API バージョン:V1、V2

親:出力

デフォルト:自動的に検出される

有効な値:0、90、180、270自動回転が必要な場合は、設定しないままにします。

互換性のあるジョブの種類: vod

例:180

説明:

ムービーを明示的に回転します。デフォルトでは、Zencoder は回転したコンテンツ (回転した iPhone ビデオを含む) を自動的に回転しますが、手動でムービーを回転することもできます。「回転」を 0 に設定すると、自動回転は行われません。自動回転が必要な場合は、このオプションはオフのままにします。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "rotate": 180
    }
  ]
}

インターレース解除

deinterlace:String

API バージョン:V1、V2

親:出力

デフォルト:検出

有効な値:オン、オフ、検出

互換性のあるジョブの種類: vod

例:オン

説明:

インターレース解除フィルタを適用するかどうかを決定します。デフォルトは「detect」です。入力ファイルがインターレースとして検出された場合、インターレース解除されます。これを「オン」に設定すると、強制的にインターレース解除（入力がインターレースされていない場合は品質が低下します）、インターレース解除を回避するには「off」に設定します。

Zencoder は、ファイルのメタデータに基づいてこの決定を行うことに注意してください。インターレースされたコンテンツが、コンテンツをインターレース解除しないエンコーダで再エンコードしたり、インターレースとしてフラグを付けたりすると、「検出」モードでコンテンツを自動デインターレースできない場合があります。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "deinterlace": "on"
    }
  ]
}

削る

sharpen:Boolean

API バージョン:V1、V2

親:出力

デフォルト:偽

有効な値:真または偽

互換性のあるジョブの種類: vod

{

ノイズ

denoise:String

API バージョン:V1、V2

親:出力

有効な値:弱い、中、強い、または最強

互換性のあるジョブの種類: vod

例:弱い

説明:

ビデオにノイズフィルターを適用します。

有効なオプションは、「弱い」、「中」、「強」、「強」、「強い」です。通常、ノイズ除去レベル、ビットレート、ソースビデオに応じて、ファイルサイズを 5% ～ 20% 縮小します。ノイズ除去を行うと、エンコードが約20％遅くなります。

「weak」' を超える値は慎重に使用する必要がありますが、アニメーションでは高レベルのノイズ除去を処理することがあります。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "denoise": "weak"
    }
  ]
}

オートレベル

autolevel:Boolean

API バージョン:V1、V2

親:出力

デフォルト:偽

有効な値:真または偽

互換性のあるジョブの種類: vod

例:真

説明:

ビデオにカラー自動レベルフィルターを適用します。このフィルターは、一部の動画で予期しない色の問題を引き起こす可能性があるため、注意して使用してください。

VP6 video_codecは自動レベルオプションをサポートしていません。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "autolevel": true
    }
  ]
}
