概要

Zencoderは、ブロードキャストおよびプロフェッショナルワークフロー用の高度なエンコード形式とコーデックのサポートを提供します。

video_codec_preset

video_codec_preset:String

API バージョン:V2

親:outputs

有効な値: avci_50、avci_100

互換性のあるジョブタイプ:すべて

例: avci_100

説明:

特定の業界標準との互換性を可能にするビデオエンコーディングプリセット。

規格:

  • avci_50 は、AVC の 10 ビット内フレームのみの圧縮です。50 メガビット/秒、高 10 イントラプロファイル、レベル 3.2、4。SMTP RP 2027:2007 を参照。
  • avci_100 は、AVC の 10 ビット内フレームのみの圧縮です。100 Mbit/s、高 4:2:2 イントラプロファイル、レベル 4.1。SMTP RP 2027:2007 を参照。
{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_preset": "avci_100",
      "video_bit_depth": 10
    }
  ]
}

format_preset

format_preset:String

API バージョン:V2

親:outputs

有効な値: ts_dvb, ts_cableLabs, ts_broadcast, as11_hd

互換性のあるジョブタイプ:すべて

例: ts_broadcast

説明:

特定の業界標準との互換性を可能にするフォーマットプリセット。サポートされているオーディオチャンネルは 2 つだけです。

規格:

  • ts_broadcast は、標準の MPEG-TS ブロードキャスト準拠ストリームを生成します。
  • ts_dvb は、デジタルビデオ放送 (DVB) に準拠したストリームを生成します。
  • ts_cablelabs は、CableLabs 準拠の MPEG-TS ストリームを生成します。
  • as11_hd は、MXF-OP1A コンテナ内の AVC イントラ 100 ビデオ、4 または 16 チャンネルのPCM オーディオです。
{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "format_preset": "ts_broadcast"
    }
  ]
}

参照:フォーマット

ts_muxrate

ts_muxrate:Integer

API バージョン:V2

親:outputs

有効な値:正の整数。1000000最大

互換性のあるジョブタイプ:すべて

例:2600

説明:

トランスポートストリームを多重化できる最大レート（kbps）。TS 形式でのみ使用します。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "format_preset": "ts_broadcast",
      "ts_muxrate": 2600,
      "video_bitrate": 2000,
      "audio_bitrate": 128,
      "audio_sample_rate": 48000,
      "audio_constant_bitrate": true
    }
  ]
}

以下も参照してください。format_preset

ts_cbr

ts_cbr:Boolean

API バージョン:V2

親:outputs

デフォルト:偽

有効な値:真または偽

互換性のあるジョブタイプ:すべて

例:真

説明:

トランスポートストリーム層でパケットをスタッフィングして CBR 出力を作成します。

これは、完全に準拠したブロードキャストストリームには必要ですが、かなり大きなファイルが作成される可能性があります。ブロードキャストTSでのみ使用します。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "format_preset": "ts_broadcast",
      "ts_cbr": true,
      "video_bitrate": 2000,
      "audio_bitrate": 128,
      "audio_sample_rate": 48000,
      "audio_constant_bitrate": true
    }
  ]
}

以下も参照してください。format_preset

ロスレスビデオ

lossless_video:Boolean

API バージョン:V2

親:outputs

デフォルト:偽

有効な値:真または偽

互換性のあるジョブタイプ:すべて

例:真

説明:

ビデオコーデックがサポートしている場合、可逆ビデオエンコーディングをイネーブルにします。

現在、Jpeg2000 (jp2) でのみサポートされています。

では使用できませんvideo_bitrate。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec": "jp2",
      "lossless_video": true
    }
  ]
}

以下も参照してください。video_bitrate
