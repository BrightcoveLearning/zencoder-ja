形式

format:String

API バージョン:V1、V2

親:出力

デフォルト:出力ファイル名、およびビデオコーデックまたはオーディオコーデックによって決定されます。それ以外の場合:mp4 (標準出力の場合); ts (セグメント出力の場合)。

有効な値:3gp、aac、ac3、ec3、flv、m4f、mj2、mkv、mp3、mp4、mxf、ogg、TS、webm、wmv

互換性のあるジョブタイプ:すべて

例: webm

説明:

使用する出力コンテナ形式。

コンテナによっては、複数の有効なファイル拡張子があることに注意してください。通常、オーディオ/ビデオファイルには「f4v」、「ogv」、「m4v」などの名前が使用され、「f4a」、「m4a」、「oga」はオーディオのみのファイルに使用されますが、これは強制されません。同様に、「f4v」と「f4a」は、Flash Videoで再生するように設計されたmp4コンテンツによく使用されますが、これは単なる慣習です。

互換性:

  • mp4 は、h264（デフォルト）、hevc、および mpeg4 ビデオと aac（デフォルト）、mp3、ac3、eac3 オーディオと互換性があります。有効な値とファイル拡張子：mp4、m4a、m4v、f4v、f4a、m4b、m4r、f4b
  • m4fはh264とhevc videoとaac、ac3、eac3オーディオに対応している。
  • oggはtheoraビデオとvorbisオーディオと互換性があります。有効な値とファイル拡張子:ogg、oga、ogv、ogx
  • webm は vp8 (デフォルト) および vp9 ビデオおよび vorbis オーディオと互換性があります。有効な値とファイル拡張子:webm
  • 3gp は、mpeg4（デフォルト）と h264 ビデオ、および aac（デフォルト）と mp3 オーディオと互換性があります。有効な値とファイル拡張子:3gp、3gp2、3g2、3gpp、3gpp2。3gp形式では、mp4ファイル拡張子を使用することもできます。
  • flv は vp6（デフォルト）と h264 ビデオ、および aac（デフォルト）および mp3 オーディオと互換性があります。有効な値とファイル拡張子:flv
  • tsはh264ビデオとaac（デフォルト）、mp3またはac3オーディオと互換性があります。有効な値とファイル拡張子:ts
  • mkvはh264ビデオとAAC（デフォルト）、MP3、ac3またはeac3オーディオと互換性があります。有効な値とファイル拡張子:mkv
  • wmv は、wmv ビデオと wma (デフォルト) または mp3 オーディオと互換性があります。有効な値とファイル拡張子:wmv、wma
  • aacはaacオーディオと互換性があり、ビデオはありません。有効な値とファイル拡張子:aac
  • mp3はmp3オーディオと互換性があり、ビデオはありません。有効な値とファイル拡張子:mp3
  • ac3 は ac3 オーディオと互換性があり、ビデオはありません。有効な値とファイル拡張子:ac3
  • ec3 は eac3 オーディオと互換性があり、ビデオはありません。有効な値とファイル拡張子:ec3
  • mj2 は jp2 ビデオと ac オーディオと互換性があります。有効な値とファイル拡張子:mj2、mjp2。
  • mxf は h264 ビデオおよび PCM オーディオと互換性があります。有効な値とファイル拡張子:mxf。
{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "format": "webm"
    }
  ]
}

video_codec

video_codec:String

API バージョン:V1、V2

親:出力

デフォルト:形式、プロファイル、または audio_codecによって決定されます。指定されていない場合は h264。

有効な値： h264、hevc、jp2、mpeg4、theora、vp6、vp8、vp9、およびwmv

互換性のあるジョブタイプ:すべて

例: vp8

説明:

使用する出力ビデオコーデック。Zencoder は現在、h264 (デフォルト)、hevc、jp2、theora、vp6、vp8、vp9、mpeg4、および wmv のコーデックをサポートしています。

互換性:

  • h264 は ac (デフォルト) および mp3 オーディオと mp4 (デフォルト)、m4f、FLV、または 3gp フォーマットと互換性があります
  • hevc は ac (デフォルト) または MP3 オーディオと mp4 または m4f フォーマットと互換性があります。
  • vp9はvorbisオーディオとwebmフォーマットと互換性がある
  • vp8はvorbisオーディオとwebmフォーマットと互換性がある
  • vp6 は mp3 (デフォルト) または ac オーディオと互換性があり、FLV 形式
  • theoraはvorbisオーディオとoggフォーマットと互換性があります
  • mpeg4 は ac (デフォルト) および mp3 オーディオ、および mp4 (デフォルト) または 3gp フォーマットと互換性があります。
  • wmv は、wma (デフォルト) または mp3 オーディオと互換性があり、wmv 形式
  • jp2 は、PCM オーディオと mj2 フォーマットと互換性があります。
{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec": "vp8"
    }
  ]
}

audio_codec

audio_codec:String

API バージョン:V1、V2

親:出力

デフォルト:形式、プロファイル、または video_codecによって決定されます。提供されていない場合は aac。

有効な値: aac、ac3、amr、eac3、mp3、pcm、vorbis、および wma

互換性のあるジョブタイプ:すべて

例: mp3

説明:

使用するオーディオコーデック。デフォルトのコーデックは、選択したビデオコーデックと形式によって異なります。

MP4 コンテナ内の MP3 オーディオは H.264 ビデオコンテンツとともに有効ですが、Quicktime では再生されません。

互換性:

  • ac は、h264 ビデオ (mp4、m4f、3gp、または flv 形式)、mpeg4 ビデオ (mp4 または 3gp 形式)、および vp6 (FLV 形式) と互換性がある
  • vorbisは互換性のあるVP8ビデオ（webm形式）とtheora Video（ogg形式）
  • wmaはwmvビデオとwmvフォーマットと互換性がある
  • mp3 は、h264 ビデオ (mp4、3gp、または flv 形式) と wmv ビデオ (wmv 形式) と互換性があります。mp3、h264、および mp4 は Quicktime では再生されません
  • pcm は h264 ビデオおよび mxf フォーマットと互換性があります。
{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "audio_codec": "mp3"
    }
  ]
}
