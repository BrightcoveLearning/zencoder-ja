ヒント

hint:Boolean

API バージョン:V1、V2

親:出力

デフォルト:偽

有効な値:真または偽

互換性のあるジョブタイプ:すべて

例:真

説明:

RTP/RTSP の MP4 ファイルのヒントを有効にします。デフォルトの mtu_sizeは 1450 バイトです。これを変更するには、 mtu_sizeオプションを使用します。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "hint": true
    }
  ]
}

mtu_size

mtu_size:Integer

API バージョン:V1、V2

親:出力

デフォルト:1450

有効な値:100から5000までの整数

互換性のあるジョブタイプ:すべて

例:2000

説明:

MP4 ヒントの MTU サイズ (バイト単位)。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "mtu_size": 2000
    }
  ]
}
