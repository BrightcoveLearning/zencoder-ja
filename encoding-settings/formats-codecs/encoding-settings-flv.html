cue_points

cue_points:Array

API バージョン:V1、V2

親:出力

有効な値:キューポイント設定を持つハッシュの配列

互換性のあるジョブの種類: vod

説明:

Zencoder は、VP6 Flash Video (FLV) ファイルへのイベントキューポイントとナビゲーションキューポイントの追加をサポートしています。FLV ファイルをエンコードすると、キューポイントは FLV ストリームおよび FLV メタデータパケットに埋め込まれます。

キューポイントには、「ナビゲーション」と「イベント」の2種類があります。ナビゲーションキューポイントを使用すると、ユーザーがビデオファイルの指定した部分を検索できます。ナビゲーションキューポイントでは、指定したキューポイント位置にキーフレームが作成されるため、コードを使用してビデオプレーヤーの再生ヘッドをその場所に移動できます。FLV ファイルで、ユーザがシークする特定のポイントを設定できます。たとえば、ビデオに複数のチャプターまたはセグメントがあり、ナビゲーションキューポイントをビデオファイルに埋め込んでビデオを制御できます。

イベントキューポイントを使用して、FLV 再生中に指定されたポイントでトリガーされる SWF ビデオプレーヤーでアクションを実行できます。

詳細については、キューポイントに関するアドビの記事を参照してください。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "cue_points": [
        {
          "type": "navigation",
          "time": 30.4,
          "name": "My Cue Point",
          "data": {
            "mykey": "myvalue"
          }
        }
      ]
    }
  ]
}

参照:type , time、name、、data
