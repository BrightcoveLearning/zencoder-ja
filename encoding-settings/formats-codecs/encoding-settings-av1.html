エンコードに関する推奨事項

AV1 は、Open Media Video Alliance of Open Media Video のオープン、ロイヤリティフリー、次世代のビデオエンコーディングフォーマットです。HEVC、VP9、H.264コーデックに比べて大幅に低いビットレートで優れた品質を提供することができます。

詳細については、 AV1ガイドを参照してください。また、現在のサポートの制限事項に特に注意してください。

このページにある設定に加えて、他の一般的なビデオ設定を含めることもできますが、次の設定は次のとおりです。いいえ現在サポートされている：

  • video_reference_frames
  • video_bframes
  • max_video_bitrate
  • quality
  • crf
  • decoder_bitrate_cap

また、解像度設定（sizewidthまたはおよびを使用するかどうかにかかわらず）height ) 幅と高さはどちらも8の倍数でなければなりません。

video_codec_level

video_codec_level:String

API バージョン:V2

親:出力

有効な値：

  • 2.0
  • 2.1
  • 3.0
  • 3.1
  • 4.0
  • 4.1
  • 5.0
  • 5.1
  • 5.2
  • 5.3
  • 6.0
  • 6.1
  • 6.2
  • 6.3

互換性のあるジョブタイプ:すべて

例:4.1

説明:

ビットレートとコーディングツリー単位 (CTU) (AV1) を制限します。

この設定は、許可された値が異なる他のコーデックにも使用できます。ビデオエンコーディングの設定を参照してください。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_level": "5.1"
    }
  ]
}

codec_profile

video_codec_profile:String

API バージョン:V2

親:出力

有効な値:main、high、professional

デフォルト:main

互換性のあるジョブタイプ:すべて

例:4.1

説明:

エンコーディングプロファイルを設定します。

この設定は、許可された値が異なる他のコーデックにも使用できます。ビデオエンコーディングの設定を参照してください。

{
  "input": "s3://zencodertesting/test.mov",
  "outputs": [
    {
      "video_codec_profile": "high"
    }
  ]
}
